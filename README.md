# "Simon Says" Game for Arduino UNO system

## :warning: TEMPORARILY NOTE:
This code is temporarily built to run on PlatformIO IDE. It'll be compatible for Arduino IDE in the future updates!


## Required components:
- LEDs (Red, Yellow, Green, Blue): 4x
- Pushbutton: 4x
- 220Ω resistors: 4x
- Passive buzzer: 1x
- Male-Male jumper wires: 14x
- Male-Female jumper wires: 4x (Optional)
- I2C 16x2 LCD Screen: 1x (Optional)
